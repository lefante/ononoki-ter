## Clone o projeto

```bash
> git clone https://github.com/lefante/ononoki-ter
```

## Instale as dependências:
Antes de executar o comando abaixo, certifique-se de estar na pasta do projeto que
você acabou de clonar !!

```bash
> bash install.sh
```

### Ligar
```bash
> npm start
```
